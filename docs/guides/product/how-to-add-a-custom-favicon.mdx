# How to add a custom favicon

In this guide, you will learn how to add a custom favicon to SnailyCADv4.

:::warning

- **Your image must match the `image/png` format.**
- **Your image must be sizes `32x32` or larger.**

:::

1. Open the `apps/client/public` folder
2. Paste your favicon image in this folder
3. Name the file `favicon.png`
4. Rebuild the client: `pnpm turbo run build --filter=@snailycad/client`
5. Restart SnailyCAD
6. All done!
