# How do I install `yarn`?

## Requirements

- [Node.js](https://nodejs.org) **(16.6+ is required.)**

## Installtion

To install `yarn`, you simply need to open the Command Prompt or Terminal with Administrator permissions. Once it has opened, run the following command:

```bash
npm install --global yarn
```

Once the command has finished, **you must restart the Command Prompt/Terminal to use the `yarn` command**
