# 10 Codes

This value defines the status of a unit. Each 10 code has a special type so SnailyCAD knows what action it should perform.

## 10 Code Types

- `Set Off Duty`: Sets the unit to off duty. This will automatically end the unit's hour log.
- `Set On Duty`: Sets the unit to on duty. This will automatically start the unit's hour log.
- `Set Assigned`: Will set the unit status to assigned. This is automatically set when a unit is assigned to a 911 call.
- `Set Status`: Will set the unit status to the selected status. Has no further actions.
- `Panic Button`: Will send a panic button alert. This is automatically set when a unit presses the panic button.
- `En Route`: When the unit is en-route.
- `On Scene`: When the unit has arrived on-scene
- `Unavailable`: When the unit is unavailable.
