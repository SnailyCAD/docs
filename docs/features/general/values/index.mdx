# Values

Values are a list of defined values that can be used to populate form fields.
These values will automatically update if they are updated in the admin dashboard.

## How to add values

To manage values, you must have any of the manage value permissions.
Once you have the permission, you can access the values by going to the
admin dashboard and selecting a value from the values section in the sidebar.

![How to add values](/features/values/how-to-add-values.jpg)
