# Qualifications

This value defines any qualifications or awards an officer or EMS/FD deputy can obtain by an admin with `ManageAwardsAndQualifications` permission.
Any unit is able to view the qualifications of any unit when hovering over their name in the Active Officers or Active EMS/FD Deputies list.
