# Business Roles

This value defines business roles, these can be used for mananging a business employee's role.

:::note
Currently there are only 3 types of permissions for a role, these are:

- `Owner`
- `Manager`
- `Employee`

Each employee can have some extra permissions such as `Can Create Posts` and `Can Register Vehicles`.
:::
