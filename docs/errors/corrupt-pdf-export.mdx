# Corrupt PDF Export

## Possible ways to fix it

### Have Chromium dependencies installed

Please have a read through the [troubleshooting](https://pptr.dev/troubleshooting) page of Puppeteer.
It includes several ways to resolve this issue for your system.

### Root user and sandbox

:::warning
You must have SnailyCAD version `1.74.0` or higher to use this option.
:::

If you're running SnailyCAD as a root user, you will need to define the following environment variable inside of your `.env` file:

```bash .env
IS_USING_ROOT_USER=true
```
